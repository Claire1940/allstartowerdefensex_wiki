## Optimasi lanjutan: menumpuk daily quests dengan sistem lain

Saat fondasimu sudah stabil, gabungkan daily quests dengan sistem terkait:

1. **Daily challenge + window event**  
   Jika event memberi hadiah untuk aktivitas/damage/summon, jadwalkan clear daily quest-mu pada periode itu.

2. **Daily quests + NPC fight quests**  
   Jika kamu farming resource evo/ascend, masukkan NPC fights setelah daily selesai agar seluruh sesi mendukung satu jalur build.

3. **Daily quests + sisa infinite/story**  
   Gunakan waktu ekstra hanya setelah nilai harian diamankan. Ini melindungi progres meskipun kamu logout lebih cepat.

4. **Daily quests + perencanaan ekonomi**  
   Belanjakan gold dan gems dengan tujuan. Jangan buru-buru belanja kosmetik jika tim utama masih kekurangan upgrade kunci.
