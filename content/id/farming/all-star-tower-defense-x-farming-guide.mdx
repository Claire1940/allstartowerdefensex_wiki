## Loop Farming Aktif vs AFK (Pakai Keduanya)

**all star tower defense x farming guide** ini bekerja paling baik saat kamu memisahkan grind menjadi dua mode: jendela main aktif dan jendela AFK.

### Jendela Aktif (Terbaik Saat Fokus Main)
- Jalankan misi event dengan drop reroll terjamin atau tinggi.
- Prioritaskan misi reset pendek dulu (seringnya per jam).
- Selesaikan misi reset lebih panjang setelahnya.
- Klaim reward lalu langsung ulangi.

### Jendela AFK (Terbaik Semalaman atau Saat Sibuk)
- Antrekan expedition memakai unit eligible terkuatmu.
- Pilih tier expedition yang benar-benar menjatuhkan reroll.
- Ulang run pada timer tersingkat yang masih layak untuk volume.

| Gaya Main | Mode Terbaik | Sesi Umum | Tujuan Utama |
|---|---|---|---|
| **Aktif** | Loop misi event | 30-120 menit | Reroll/jam maksimum |
| **AFK** | Expeditions | 2-10 jam | Akumulasi pasif |
| **Hybrid** | Event + refresh expedition | 1-4 jam | Keseimbangan terbaik antara kecepatan dan kenyamanan |

Loop hybrid adalah titik paling ideal:
1. Mulai expedition.
2. Jalankan 1–2 misi event cepat.
3. Klaim expedition.
4. Mulai ulang expedition.
5. Ulangi.

Ini membuat akunmu menghasilkan secara paralel, yang menjadi alasan utama pemain veteran unggul jauh dalam progres trait.

## Setup Build untuk Farming Lebih Cepat

Kecepatan farming-mu sangat bergantung pada konsistensi clear. Rute sehebat apa pun akan gagal jika timmu tidak bisa menyelesaikan misi dengan cepat.

### Prinsip Setup Unit
- Gunakan unit upgrade terkuatmu untuk scaling expedition.
- Bawa AoE yang andal untuk kontrol wave di misi event pendek.
- Sisakan satu slot fleksibel untuk utilitas (slow, tanking, atau cakupan range).
- Hindari lineup terlalu spesifik yang hanya bekerja di satu map.

### Checklist Praktis Sebelum Sesi
- Klaim semua misi yang sudah selesai terlebih dahulu.
- Cek reward daily challenge.
- Verifikasi status stok shop mingguan.
- Refresh papan event dan urutkan berdasarkan reset tersingkat.
- Antrekan expedition sebelum masuk match aktif.

| Masalah | Penyebab Umum | Solusi |
|---|---|---|
| Reroll sedikit setelah sesi panjang | Farming mode nilai rendah duluan | Ubah urutan rute: shop → misi terjamin |
| Gain AFK terasa lemah | Tier expedition salah atau setup unit lemah | Susun ulang tim expedition, pilih tier yang drop reroll |
| Burnout dari grind panjang | Tidak ada rencana sesi | Jalankan blok 45–60 menit dengan tujuan jelas |
| Kecepatan clear event buruk | Tim kurang kontrol wave | Tambah AoE awal + core DPS yang stabil |
