## Checklist de solución de problemas para recompensas faltantes o retrasadas

No todo problema es elegibilidad de cuenta. A veces las recompensas parecen faltar por el flujo de reclamación, confusión de UI o retrasos de sincronización de sesión.

| Problema | Causa probable | Solución |
|---|---|---|
| Recompensa no visible tras subir de nivel | UI no actualizada | Volver a entrar al servidor y revisar de nuevo el registro de recompensas |
| Reclamada pero no en inventario | Sincronización retrasada | Esperar un poco y luego reloguear |
| Cosmético legacy faltante | Cuenta no elegible para esa ventana anual | Verificar la línea de tiempo de actividad histórica |
| Título no disponible | Condición de juego temprano no cumplida | Confirmar si es específico OG, no basado en nivel |
| La progresión se siente lenta | Selección de modo ineficiente | Cambiar a clears repetibles con menor tasa de fallos |

### Flujo de diagnóstico rápido

1. Confirma qué tipo de recompensa falta (nivel, evento u OG legacy).  
2. Verifica tu último hito de nivel completado y el paso de reclamación.  
3. Vuelve a entrar y revisa otra vez las pestañas de inventario/perfil.  
4. Compara el historial de tu cuenta con los requisitos de ventana legacy.  
5. Si no se resuelve, envía un ticket de soporte conciso con pruebas.

Este flujo de trabajo resuelve la mayoría de los problemas percibidos con las **recompensas de nivel de All Star Tower Defense X** sin adivinanzas.
