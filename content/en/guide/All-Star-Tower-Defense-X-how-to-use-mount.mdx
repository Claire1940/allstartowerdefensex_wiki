## Full Mount Setup Walkthrough (Beginner to Reliable)

When players ask for **All Star Tower Defense X how to use mount**, the missing part is usually not “equip,” but “activate.” Equipping only prepares the mount. You still need to call it from the wheel.

### 1) Equip from the Items Menu
Go into your item inventory and locate the mount category. Select the mount you want (for example, cloud-style mounts), then hit **Equip**.

### 2) Assign a Wheel Slot
After pressing equip, assign the mount to one of the wheel slots. This matters if you own multiple mounts, because each one needs a slot to become quickly selectable.

### 3) Open the Mount Wheel
Press **`.` (period/full stop)** on keyboard. This opens the wheel UI where your assigned mounts appear.

### 4) Summon the Mount
Pick the mounted slot you assigned. Your character should immediately summon it.

### 5) Dismiss or Toggle Off
Press **`.`** again and click the active mount to dismiss it.

Here’s the same process with expected outcomes and common failure points:

| Phase | What You Should See | Common Mistake | Fix |
|---|---|---|---|
| **Equip** | Mount details + Equip button | Player exits menu too soon | Confirm slot assignment first |
| **Slot Assign** | Wheel slot highlight | Slot not saved | Re-open item and assign again |
| **Wheel Open** | Circular mount menu | Wrong key pressed | Use `.` key based on your keyboard layout |
| **Summon** | Character mounts instantly | Clicking empty slot | Choose slot that contains your mount |
| **Dismiss** | Return to normal movement | Pressing random keybind | Re-open wheel and click active mount |

## Troubleshooting: Why Your Mount Might Not Work

If your **All Star Tower Defense X how to use mount** setup still fails, run this checklist in order:

### Quick Diagnostic Checklist
- Is the mount actually equipped from Items?
- Did you assign it to a wheel slot?
- Are you pressing the period key (`.`), not comma or slash?
- Is chat box active (capturing keyboard input)?
- Did your keyboard layout change (US vs non-US)?
- Are you trying to click an empty wheel slot?
- Did a recent patch reset some UI assignments?

### Fast Fix Matrix

| Problem | Likely Cause | Fix Time | Best Fix |
|---|---|---|---|
| **Wheel won’t open** | Wrong key/input capture | 10-30 sec | Exit chat, press `.`, retest |
| **Mount not appearing in wheel** | No slot assignment | 20-40 sec | Re-equip and assign a slot |

| **Works in lobby, not in match** | Mode/UI restrictions | 30-60 sec | Test in allowed phase or relaunch |
| **Intermittent failure** | Keybind conflict/layout issue | 1-3 min | Reset keybinds and verify keyboard layout |

If needed, relog after reassigning slots. In many cases, a clean reconnect refreshes input/UI state.
